# gRPC

* gRPC 由Google开发的远程过程调用(RPC)系统
* 语言中立、平台中立、开源的

## [官方文档](https://grpc.io/)

	[NodeJS支持](https://grpc.io/docs/languages/node/)
	
## HTTP 2.0

* 在2013年8月进行首次合作共事性测试
* 只用于https
* 基本主流浏览器都已支持
* 特点和优势
	* 多路复用请求
	* 对请求划分优先级
	* 压缩HTTP头
	* 服务器推送流（即Server Push技术）
	* SPDY试图保留HTTP的现有语义，所以cookies、ETags等特性都是可用的
